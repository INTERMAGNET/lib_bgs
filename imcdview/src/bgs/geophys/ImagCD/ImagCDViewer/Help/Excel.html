Intermagnet CD Data Viewer Help

Microsoft Excel

If you are running the data viewer on Microsoft Windows then you can copy data from the program to Microsoft Excel using the clipboard. To do this:
  • Bring up the Data Viewer (for example, click on an observatory in the World Map window and select 'View Data' from the pop-up menu).
  • Select the 'View Data' tab - you should see a table of values.
  • Select the area that you want to copy: First click the mouse in the cell at the top left of the area that you want to copy. Next move to the bottom, press the 'Shift' key on your keyboard and click the mouse in the cell at the bottom right. The area that you are going to copy should now be highlighted.
  • Press the 'Copy' button or type CTRL/-C on your keyboard.
  • Move to Excel and paste the data into a worksheet.
