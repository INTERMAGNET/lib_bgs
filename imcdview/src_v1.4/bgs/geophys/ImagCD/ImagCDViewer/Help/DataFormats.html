Intermagnet CD Data Viewer Help

Data Formats

The INTERMAGNET CD Data Viewer reads data in INTERMAGNET archive format. It can export data in IAGA 2002 format, World Data Centre format (minute and hourly data only) and INTERMAGNET Minute Mean format (minute data only). With the exception of the World Data Centre format, all these data formats are described in the INTERMAGNET technical manual and on the INTERMAGNET web site. The World Data Centre format is not described on the INTERMAGNET web site - details of the format can be found in a separate page of the CD Data Viewer help system.
