Intermagnet CD Data Viewer Help

New User

The information on this page is for people who have not used the Intermagnet Data Viewer software before. Most of the software is intuitive and easy to use but you will need to know how to configure the software to find your data before you can use any of the functions. To read data from a CD-ROM or DVD take the following steps:

  1. Insert the CD-ROM or DVD in your computer's reader.
  2. Display the 'Select Data Sources' dialog by clicking here (the normal way to find this dialog is from the 'Database' menu on the program's main window).
  3. Press the 'Add' button on the 'Select Data Sources' dialog. A directory selection dialog is displayed.
  4. If you are using Microsoft Windows: There is a drop down list at the top of the directory selection dialog - choose the drive letter that corresponds to your CD-ROM or DVD drive (e.g. "D:\").
  5. If you are using UNIX: Use the directory selection dialog to navigate to the directory where your CD-ROM or DVD drive is mounted.
  6. On the directory selection dialog press 'Select directory'.
  7. On the 'Select Data Sources' dialog press 'OK'.
  8. You should see observatories plotted as red dots over the world map in the main window.

The help pages contain further information about databases and data sources. Also see Quick start
